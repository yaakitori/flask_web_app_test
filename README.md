# Flask書籍管理アプリ

これはFlaskとSQLite3を使用して作成された、シンプルな書籍管理ウェブアプリケーションです。書籍情報の登録、表示、編集、削除（CRUD）の基本的な機能を実装しています。

## 主な機能

-   **書籍一覧表示**: 登録されている書籍を一覧で表示します。
-   **新規登録**: 新しい書籍の入荷日、タイトル、金額を登録できます。
-   **情報編集**: 登録済みの書籍情報を更新できます。
-   **情報削除**: 登録済みの書籍を選択して削除できます。

## ディレクトリ構成

このアプリケーションの主要なファイル構成は以下の通りです。

```
├── database.db         # SQLiteデータベースファイル
└── flaskr              # Flaskアプリケーションパッケージ
    ├── __init__.py     # アプリケーションの初期化、DBのセットアップ
    ├── db.py           # データベーステーブル作成用モジュール
    ├── main.py         # ルーティング、ビュー関数の定義
    ├── static
    │   └── style.css   # アプリケーションのスタイルシート
    └── templates
        ├── index.html      # 書籍一覧ページ
        ├── form.html       # 新規登録ページ
        ├── edit.html       # 編集ページ
        └── delete.html     # 削除ページ
```

## 技術スタック

-   **フレームワーク**: Flask
-   **データベース**: SQLite3
-   **フロントエンド**: HTML/CSS

## セットアップと実行方法

1.  **必要なライブラリをインストールします。**
    ```bash
    pip install Flask
    ```

2.  **Flaskアプリケーションを実行します。**
    ターミナルで以下のコマンドを実行してください。
    ```bash
    flask --app flaskr run
    ```
    *アプリケーションの初回実行時に、`database.db`ファイルと`books`テーブルが自動的に作成されます。*

3.  **ブラウザでアクセスします。**
    ウェブブラウザを開き、以下のURLにアクセスしてください。
    `http://127.0.0.1:5000`
